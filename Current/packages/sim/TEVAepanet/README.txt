The EN2source.zip file contains the original source code as downlaoded from EPA's website at:

http://www.epa.gov/nrmrl/wswrd/dw/epanet.html#downloads

The src directory contains the EPANet source code with modifications to enable it to work with TEVA.  Assurances have been made that these changes will be in future releases so the original code won't need to be updated woth our changes anymore.
