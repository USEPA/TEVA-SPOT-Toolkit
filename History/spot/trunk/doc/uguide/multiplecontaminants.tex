\section{Usage}\label{multiplecontaminants_multiple_contaminantsUsage}
The following is an example showing how to do a single sensor placement optimization that considers an ensemble of events involving multiple contaminants. Assume that there are two TSG files: one describing a set of events associated with a chemical contaminant, and one describing a set of events associated with a biological contaminant. Assume also that there are corresponding TAI files describing the dose-\/response properties of these contaminants.

First generate two TSO files: one for each TSG file. We'll assume that we are working on the test1 example in sp/test. Call these TSO files {\bfseries test1b.tso} and {\bfseries test1c.tso}. Now, we will process these using {\bfseries tso2Impact}

\begin{verb}
   tso2Impact --detectionLimit 300 --detectionLimit 0.00001 --pe test1 test1b.tso test1c.tso test1b.tai test1c.tai
\end{verb}


The result is a single impact file {\bfseries test1\_\-pe.impact} that can be treated like any other impact file. For example, we can run any of the scripts in packages/sp/test that expect a problem named {\bfseries test1}. For example: \begin{verb}
   ../../../bin/sp --path=../../../bin --network=test1 --objective=mc_mean --ub=ns,5 --solver=snl_grasp
\end{verb}
