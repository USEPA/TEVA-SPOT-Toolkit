\section{Overview}\label{evalsensorExecutable_evalsensorOverview}

The {\bfseries evalsensor} executable is used to compute information about the impact of contamination events for one (or more) sensor placements.

\section{Description}\label{evalsensorExecutable_evalsensorDescription}

The {\bfseries evalsensor} executable takes sensor placements in a \doxyref{Sensor Placement File}{p.}{formats_sensorPlacementFile} and evaluates them using data from an \doxyref{Impact File}{p.}{formats_impactFile} (or a list of impact files). This executable measures the performance of each sensor placement with respect to the set of possible contamination locations.

See \doxyref{Section 5.9}{p.}{solvers_solvers6} for further description of this command.
