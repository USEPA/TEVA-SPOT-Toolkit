This document describes the TEVA-\/SPOT Toolkit (SPOT). SPOT provides a sensor placement framework that facilitates research in sensor placement optimization and enables the practical application of sensor placement solvers to real-\/world water security applications. SPOT contains a number of command line executables that can be used to analyze the impact of contamination events and to identify optimal sensor placements. \\\cline{1-1}
