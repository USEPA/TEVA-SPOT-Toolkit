TEVA_SPOT README
================

The TEVA_SPOT Python package integrates a variety of Python utilities that
are used in the TEVA-SPOT Toolkit.


INSTALLATION
============

See the INSTALL.txt file.


LICENSE
=======

See the LICENSE.txt file.


GETTING STARTED
===============

Directories

   teva_spot - The root directory for the Python source code
   scripts   - Contains command-line executables
   test      - Directories of test utilities

Documentation and Bug Trcking

    The TEVA-SPOT Toolkit Trac wiki supports documentation, issue tracking,
    and browsing of the subversion repository that includes this package:

        https://software.sandia.gov/trac/spot

THIRD PARTY SOFTARE
===================

Some utilities in the TEVA_SPOT Python package depend on executables that are built within the TEVA-SPOT Toolkit.  Testing TEVA_SPOT depends on the 'nose' package.
