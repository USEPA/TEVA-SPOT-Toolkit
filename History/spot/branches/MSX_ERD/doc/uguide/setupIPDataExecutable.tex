\section{Overview}\label{setupIPDataExecutable_setupIPDataOverview}

The {\bfseries setupIPData} executable is used by the {\bfseries sp} solver interface to setup an integer programming formulation for sensor placement.

\section{Description}\label{setupIPDataExecutable_setupIPDataDescription}

The input file used by {\bfseries setupIPData} is a \doxyref{Sensor
Placement Configuration File}{p.}{formats_spConfigFile}. The output of
this executable is sent to standard out, and it is in a format that
can be processed by \doxyref{PICO}{p.}{picoExecutable} and the AMPL
solver interface.

\section{Notes}\label{setupIPDataExecutable_setupIPDataNotes}

\begin{itemize}
\item This executable is not meant to be run interactively. 
\item The scalability of this solver has not been well-\/characterized for large datasets, even when using aggressive aggregation. 
\end{itemize}
