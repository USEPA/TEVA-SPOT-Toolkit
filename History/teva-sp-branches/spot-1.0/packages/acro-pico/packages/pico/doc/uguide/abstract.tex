This document provides a user guide for the PICO software library.
PICO (Parallel Integer and Combinatorial Optimizer) is an object-oriented
framework for parallel implementation of branch-and-bound algorithms.
Using object-oriented techniques, PICO's parallel search engine is cleanly
separated from the details of the application and computing platform. This
approach allows the same underlying parallel search code to be used on
a wide variety of branch-and-bound applications.  The PICO software library
can be used in several ways: (1) as a linear or integer programming
solver launched by AMPL, (2) as a stand-alone linear or integer programming
solver, (3) as a framework for developing customized integer programming
solvers and (4) as a general-purpose framework for serial and parallel
branching algorithms.  This user guide illustrates each of these uses
of PICO.
